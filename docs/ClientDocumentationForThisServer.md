# Documentation for clients for this server

## Defaults commands
Here are the commun commands that are accessibles to you (locals commands).

- > End
    
    Close the communication with the server.

- > Help

    Give you all the commands you can have access to.

- > Get \<pathname\>

    Get the file at this path (relative path)

- > List \<pathname\>

    List everything that this path (relative path) contain.
