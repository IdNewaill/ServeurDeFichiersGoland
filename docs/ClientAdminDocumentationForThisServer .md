# Documentation for clients for this server

## Defaults commands
Here are the commun commands that are accessibles to you (locals commands).

Note that the symbol ⛉ mean that this is an administrator command only !

- > End
    
    Close the communication with the server.

- > Help

    Give you all the commands you can have access to.

## File Related Commands
Commands that are using or have a link with files

Note that the symbol ⛉ mean that this is an administrator command only !

- > List \<pathname\>

    List everything that this path (relative path) contain.

- > Hide \<pathname\>

    ⛉ Hide this file to everybody until you show it again using the command "**Reveal \<pathname\>"

- > Reveal \<pathname\>

    ⛉ Let this file visible to everybody until you hide it again using the command "**Hide \<pathname\>"

## Server Related Commands
Commands that are using or have a link with files

Note that the symbol ⛉ mean that this is an administrator command only !

- > Terminate \<pathname\>

    ⛉ Let this file visible to everybody until you hide it again using the command "**Hide \<pathname\>"
